Formulário
de Requisitos

Um formulário de requisitos é um documento utilizado no processo de desenvolvimento de software (ou outros projetos) para coletar, organizar e formalizar as necessidades e expectativas dos usuários, clientes ou stakeholders. Ele serve como uma base para garantir que todos os envolvidos no projeto tenham uma compreensão clara e compartilhada do que será desenvolvido ou entregue.

Plano
de trabalho

Um plano de trabalho é um documento em que se registram as atividades, os objetivos e as tarefas necessárias para a performance de um profissional. Sua importância está diretamente relacionada com o alcance de propósitos.

Game design
document

O GDD (ou Game Design Document) é um documento contendo todas as informações relevantes do design de um jogo: temática, mecânicas, plataformas, inimigos, levels, entre outros.
